# Seller Risk & Fraud Trend Analysis Dashboard
## Project Overview

This project analyzes transaction behavior and fraud indicators within an e-commerce environment to identify operational risk trends, suspicious transaction patterns, and key fraud drivers using SQL, Excel, and Power BI.

The analysis focuses on:
- Fraud detection trends
- Geographic risk indicators
- Transaction behavior analysis
- Authentication failure patterns
- Operational KPI reporting

---

## Tools Used

- SQL
- Microsoft Excel
- Power BI
- GitHub

---

## Key Business Questions

- Which transaction patterns are most associated with fraud?
- Which countries and merchant categories present the highest risk?
- How do authentication signals impact fraud rates?
- What operational trends can improve fraud detection efficiency?

## Dataset

Source: Kaggle E-Commerce Fraud Detection Dataset
